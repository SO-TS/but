# but

一个文件夹实时备份工具，使用 *Rust* 编写。

该程序是 [WSMCS 服务器](https://www.wsmcs.top) 工具的一部分。

## 使用

您可以手动克隆项目到本地进行编译。也可在 GitHub Actions 下载自动编译制品。

在第一次使用时，您需要输入该命令生成配置文件：

`but --init`

修改配置文件即可使用。
